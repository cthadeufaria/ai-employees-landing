# Drinkerita AI Employees Landing

Landing page estática para apresentar o serviço de criação de funcionários de IA e o case do ERP da Drinkerita.

## Rodar localmente (acesso pela rede local)

No diretório do projeto:

```bash
python3 -m http.server 8080 --bind 0.0.0.0
```

Depois abra:

- No seu computador: `http://127.0.0.1:8080`
- Na rede local: `http://192.168.1.109:8080`

## Deploy separado

Como é um site estático, você pode publicar em Render Static Site, Netlify, Vercel ou GitHub Pages.
Apenas envie os arquivos `index.html` e `styles.css`.
